# Resource - GET

Return an individual currency.

## Request

**URL** : `/v3/currencies/{currency_id}`

**Method** : `GET`

**Parameters** : None

## Response

**Code** : `200 OK`

**Content** : 
```json
{
    "id": "epMqeYqPkL",
    "code": "GBP",
    "name": "Sterling",
    "created": "2020-09-28 10:27:34"
}
```
